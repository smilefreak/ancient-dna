# Installation

## Debian linux (Ubuntu)
```
wget -qO- https://deb.nodesource.com/setup_4.x | sudo bash -  
sudo apt-get install --yes nodejs  
npm install  
```

## OSX 

installation using homebrew.
```
brew install node 
```

# Testing

Navigate to web directory and run node.

```
cd web 
node start
```
